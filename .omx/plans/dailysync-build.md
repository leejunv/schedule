# DailySync build plan

## Goal

Build a responsive Next.js App Router productivity web app named DailySync with calendar, tasks, recurrence, habits, local persistence, import/export, dark mode, and PWA support.

## Slices

1. Scaffold Next.js + TypeScript + Tailwind app structure.
2. Add domain types, recurrence utilities, date helpers, and Zustand store with localStorage persistence.
3. Build responsive UI: calendar, day panel, quick task input, filters, summary, habits, data tools, and mobile tabs.
4. Add PWA manifest/service worker and install metadata.
5. Verify with install/build and record outcome.

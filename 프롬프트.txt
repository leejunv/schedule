날짜별 일정관리 및 반복 체크리스트 웹앱을 만들어줘.

목표:

* PC와 모바일에서 모두 사용 가능한 생산성 웹앱 제작
* 캘린더 + 체크리스트 + 반복 일정 + 습관 관리 기능 통합
* 직관적이고 빠른 입력 UX 제공
* 실제 서비스 수준의 구조로 설계

기술 스택:

* React + TypeScript 기반
* Next.js(App Router) 사용
* Tailwind CSS 사용
* Zustand 또는 Context API 상태관리
* LocalStorage 기반 저장
* 추후 Firebase 연동 가능한 구조로 설계
* 반응형 웹 디자인 필수
* PWA 지원

디자인 컨셉:

* Notion + Google Calendar + Todoist + TickTick 스타일
* 미니멀하고 현대적인 UI
* 생산성 중심 UX
* 카드 기반 레이아웃
* 모바일 터치 최적화
* 다크모드 / 라이트모드 지원
* 부드러운 애니메이션 적용
* 접근성 고려

핵심 기능:

1. 날짜별 일정 관리

* 월간 캘린더 UI 제공
* 날짜 클릭 시 해당 날짜 일정 표시
* 오늘 날짜 강조
* 날짜별 일정 개수 표시
* 날짜별 완료율 표시
* 날짜별 메모 작성 가능

2. 체크리스트 기능

* 할 일 추가 / 수정 / 삭제
* 체크 완료 기능
* 드래그 앤 드롭 정렬
* 우선순위 설정

  * 높음
  * 중간
  * 낮음
* 카테고리 태그 기능
* 완료된 항목 숨기기
* 빠른 입력 기능 제공

3. 반복 일정(Recurring Tasks) 기능
   반복 일정은 매우 쉽게 생성 가능해야 함.

반복 옵션:

* 매일
* 매주
* 매월
* 매년
* 평일만
* 사용자 지정 반복

사용자 지정 반복:

* n일마다 반복
* 특정 요일 반복
* 매월 특정 날짜 반복
* 종료일 설정 가능
* 반복 횟수 제한 가능

반복 일정 UX:

* 일정 생성 시 “반복 설정” 버튼 제공
* 자주 사용하는 반복 옵션은 원클릭 제공
* 모바일에서는 바텀시트 UI 사용
* 반복 일정은 아이콘 표시

반복 일정 로직:

* recurrenceRule 구조 사용
  예시:
* DAILY
* WEEKLY:MON,WED,FRI
* MONTHLY:15

추가 기능:

* 특정 일정만 수정
* 전체 반복 일정 수정
* 반복 일정 자동 생성
* 완료 못한 일정 자동 이월 옵션
* 반복 일정 필터 제공

4. 습관 추적(Habit Tracker)

* 반복 일정과 연동 가능
* 연속 달성(Streak) 표시
* 주간/월간 달성률 제공
* 습관 통계 UI 제공

5. 일정 요약 기능

* 오늘 해야 할 일 요약
* 완료율 Progress Bar
* 남은 일정 표시
* 우선순위 높은 일정 상단 표시

6. 검색 및 필터

* 키워드 검색

* 새로고침 후 데이터 유지
* JSON Export / Import 기능
* 추후 클라우드 동기화 가능한 구조

8. 모바일 UX

* 완전한 반응형 지원
* 하단 탭 네비게이션
* 터치 친화적 인터페이스
* 모바일 캘린더 최적화
* 스와이프 인터랙션 지원

9. 추가 고급 기능

* 알림(Reminder) 구조 포함

* AI 일정 추천 기능 확장 가능하도록 설계

10. 성능 최적화

* 캘린더 렌더링 최적화
